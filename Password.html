What is a Password

A password by definition is a secret phrase or string of characters which is used to identify the user during the authentication process.

Strong password Advice

To create a strong password it shouldn't be something that can't be guessed to easily such as your birthday,your name or country.

  • The longer the password obviously the more secure it is but 12 charcaters minimum is recommended
  • Use different numbers,letters and symbols throughout your Password
  • Don't do obvious letter or word combination like 123456,abc,asdfghjkl; as they are too easy to guess
